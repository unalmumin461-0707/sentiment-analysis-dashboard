# Sentiment Analysis Dashboard

## Project Description
This project performs sentiment analysis on customer reviews using machine learning techniques.

## Dataset
Amazon Reviews Dataset

## Models Used
- Logistic Regression
- Naive Bayes
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Technologies
- Python
- Scikit-learn
- SQLite
- Streamlit

## Files
- app.py
- sentiment_model.pkl
- vectorizer.pkl
- predictions.db
